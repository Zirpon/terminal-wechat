# terminal-wechat

> this is a terminal wechat, supported send and receive wechat group or friend Text msg, in the python interactive mode

usage:

```shell
python
>>> import terchat
>>> terchat.start()
>>> terchat.send_msg("蛋挞 我要回家了", "@92c3dc453017be5ef25908102a925cd1")
>>> terchat.end()
>>> quit()
```
